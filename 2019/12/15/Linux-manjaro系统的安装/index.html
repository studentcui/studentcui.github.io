manjaro系统的安装

Posted on 2019-12-15 Edited on 2020-06-05 In Linux

https://distrowatch.com/ linux系统交流网站

安装

官网下载manjaro ios镜像

软碟通刻录u盘，写入方式选raw

u盘启动

tz timezone 时区 asia/shanghai

keytable 键盘 us

lang 语言 us

driver 驱动，nofree不用自己装

boot启动

Welcome to Manjaro

最下方有 Launch installer

安装注意！分区自己分

双系统把 /boot/efi 挂载在win10 的引导上

然后给linux的所有空间挂载为 / 即可

安装完成后

改国内源

sudo pacman-mirrors -i -c China -m rank

sudo pacman -Syy

选最快的源即可

有的软件pacman找不到，所以推荐修改GPG key，即可用pacman安装常用软件

修改/etc/pacman.conf，在最后一行加

然后执行

sudo pacman -S archlinuxcn-keyring

sudo pacman -Syy

禁用关机按钮蜂鸣声

在/etc/modprobe.d/中创建.conf文件，使用blacklist关键字屏蔽不需要的模块，这里是pcspkr模块。

文件中写

blacklist命令将屏蔽一个模块，所以不会自动装入，但是如果有其他非屏蔽模块需要这个模块，系统仍然会装入

（可选）

为了不让系统因为依赖装入，

sudo vim /etc/modprobe.d/blacklist.conf

在文件中

常用软件

yay或aurman

可以代替pacman安装软件

sudo pacman -S yay base-devel

搜狗输入法

在～/.xprofile中添加

重启

Google Chrome

sudo pacman -S google-chrome

Vim

sudo pacman -S vim

wps

sudo pacman -S wps-office

sudo pacman -S ttf-wps-fonts

网易云音乐

sudo pacman -S netease-cloud-music

QQ

sudo pacman -S deepin.com.qq.im

深度截图

sudo pacman -S deepin-screenshot

时间改为北京时间

timedatectl set-local-rtc 1 --adjust-system-clock 把rtc时间解释成本地时间

以下二选一

sudo hwclock -s localtime读硬件时间写入系统时间，读硬件时间解释为本地时间

或

sudo timedatectl set-local-rtc true将硬件时钟设置为本地时区

优质字体

飞机

安装electron-ssr

打开terminal

export http_proxy="http://127.0.0.1:12333"

export https_proxy="http://127.0.0.1:12333"

然后crul www.google.com如果返回数据代表成功

chrome安装SwitchOmega插件

profiles全删掉

添加两个

  1. 名称：ssr 类型：proxy profile
  2. 名称：auto 类型：switch profile

设置ssr 代理协议 http 代理服务器 127.0.0.1 代理端口 12333

设置auto

Switch rules添加ssr

Rule List Config

Rule List format选Switchy

Rule List URL https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt

左下 Apply changes

打开飞机，然后右上角插件选中ssr就可以用了

全面详细的文档 https://wiki.archlinux.org
