Linux

Linux通用知识

常用

pwd 显示当前目录

ls显示当前目录文件

File

复制 cp

cp [option] source1 source2 source3 … directory

-a archive，复制所有目录

-i 若目标文件存在，询问是否覆盖

cp /test1/file1 /test2/file2 将file1复制为file2

移动 mv

mv [-fiv] source destination

-i 若目标文件存在，询问是否覆盖

mv /test1/file1 /test2/file2 将file1移动为file2

飞机

ubuntu系统里只有python3 没有python 所以用不了electron-ssr 建立一个软连接就能用了
sudo ln -s /usr/bin/python3 /usr/bin/python

screen 多重视窗管理程序

非root用户使用screen需要先运行script /dev/null

  • screen -S sessionName command sessionName为会话名称 command为想要执行的命令

  • Ctrl+a d 进入detached状态，暂时离开正在运行的session，后台运行

  • screen -ls 查看所有会话

  • screen -r sessionName 恢复被detached的会话

  • screen -X -S sessionName quit杀死一个已经detached的会话
