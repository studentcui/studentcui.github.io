Git提交流程

git pull取回服务器代码，与本地合并

git status查看当前项目中哪些文件被修改过

Untracked: 未跟踪,一般为新增文件，此文件在文件夹中, 但并没有加入到git库, 不参与版本控制. 通过git add 状态变为Staged.

Modified: 文件已修改, 仅仅是修改, 并没有进行其他的操作.

deleted： 文件已删除，本地删除，服务器上还没有删除.

renamed：

git add 文件

git commit -m "提交信息"

git push
